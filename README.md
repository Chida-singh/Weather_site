# Atmosphere | Weather App

Live Demo: https://chida-singh.github.io/Weather_site/

## Overview
Atmosphere is a minimal and responsive weather application that provides real-time weather updates for any city worldwide. It fetches data from WeatherAPI and dynamically updates the interface with current weather conditions such as temperature, humidity, and wind speed.

## Features
- Search weather for any city worldwide
- Displays real-time temperature, humidity, wind speed, and conditions
- Dynamic background changes based on weather
- Responsive design for desktop and mobile
- Simple and user-friendly interface

## Tech Stack
- HTML5
- CSS3
- JavaScript
- WeatherAPI
